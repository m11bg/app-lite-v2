# Contexto do Projeto: Marketplace de Serviços

## Sobre o Projeto
Este é um aplicativo de marketplace de serviços construído com TypeScript, React e Expo (React Native), visando as plataformas Web, Android e iOS. A arquitetura segue os princípios de componentização e estado gerenciado.

## Stack de Tecnologia
- **Linguagem:** TypeScript
- **Frontend Web/Mobile:** React com React Native (Expo)
- **Estilização:** Tailwind CSS (ou sua biblioteca de preferência)
- **Gerenciamento de Estado:** Redux Toolkit (ou sua biblioteca de preferência)

## Padrões de Código
- Use componentes funcionais com Hooks.
- Dê preferência a interfaces (interface) para definir props de componentes.
- Todo código deve ter tipos explícitos. Evite `any`.
- Nomenclatura de arquivos de componente: `NomeComponente.tsx`.
